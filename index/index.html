Hi, there!

Here is rustech, a web server which packages some important APIs of the TIS server of SUSTech. Here are some tips for you to use it quickly!

1. "/cas_login?username=&password=" test whether a account is available.

2. "/basic_info?username=&password=" Obtain basic information of a student.

3. "/semester_gpa?username=&password=" Get the GPA of all semester for specific student.

4. "/courses_grades?username=&password=" Obtain grades of all the courses for specific student.

5. "/courses" Obtain all the courses from the TAO.

6. "/selected_courses?username=&password=&semester_year=&semester_no=" Obtain all the selected courses of specific student.

7. "/available_courses?username=&password=&semester_year=&semester_no=&courses_type=" Get the selectable courses for specific student.

8. "/select_course?username=&password=&semester_year=&semester_no=&course_id=&course_type=&points=" Select a class for student.

9. "/drop_course?username=&password=&semester_year=&semester_no=&course_id=" Drop a class for student.

10. "/update_points?username=&password=&semester_year=&semester_no=&course_id=&points=" Update the points of specific selected class for a student.

11. "/course_outline?username=&password=&outline_id=" Get the outline of a class.

12. "/course_table?username=&password=&semester_year=&semester_no=" Get the course table of specific student

For more information, you can visit https://github.com/gdjs2/rustech.
